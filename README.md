# 🍀 Día de mi Suerte — Landing Page

Landing page de alta conversión para [diademisuerte.com](https://diademisuerte.com).

## Stack
- **Next.js 16** (App Router) + **TypeScript** + **Tailwind CSS**

## Desarrollo
```bash
npm install
npm run dev
```

## Deploy
Conecta el repositorio a [Vercel](https://vercel.com) para deploy automático.

## Personalización
- **Premio semanal:** edita `THIS_WEEK_PRIZE` en `src/components/Landing.tsx`
- **Fotos ganadores:** coloca en `public/winners/` y descomenta `<img>` tags
- **Videos reels:** coloca en `public/reels/` y descomenta `<video>` tags
- **Productos:** coloca en `public/products/` y reemplaza emojis del hero
- **WhatsApp:** cambia `51XXXXXXXXX` por tu número real
